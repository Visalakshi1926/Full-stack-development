CSS - Cascading Style Sheet


Types of CSS

  • Inline CSS - Style inside the html document
  • Internal CSS - Style inside html using a style tag inside head tag
  • External CSS - Styling the webpage by including a separate file
  1. Element - selecting each tag
  2. Class and Id - selecting particular tag
  3. Grouping - styling for more than one tag
  4. Universal - Styling entire tag
  5. Attribute - used inside a property:value pair tag
  6. Child and decent - style inside a nested tag
